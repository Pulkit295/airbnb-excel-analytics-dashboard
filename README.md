# 🏠 Airbnb Analytics Dashboard using Microsoft Excel

## 📌 Project Overview

This project presents an end-to-end data analysis of an Airbnb dataset using Microsoft Excel. The objective was to clean the data, perform exploratory data analysis (EDA), engineer meaningful features, and build an interactive dashboard that provides actionable business insights.

The project demonstrates Excel skills commonly required for Data Analyst roles, including data cleaning, Pivot Tables, Pivot Charts, Conditional Formatting, Slicers, and Dashboard Design.

---

## 🎯 Project Objectives

- Clean and prepare raw Airbnb listing data
- Perform comprehensive data profiling
- Conduct Univariate, Bivariate, and Multivariate Analysis
- Create meaningful analytical features
- Develop an interactive Excel dashboard
- Generate business insights from the data

# 📈 Key Business Insights

- Average Airbnb listing price is **177.18**.
- Most listings fall within the **101–200** price range.
- Entire homes have the highest average price among common room types.
- Private rooms provide more budget-friendly accommodation.
- Pricing varies significantly across neighbourhoods.
- Availability patterns differ by listing type.
- Interactive filters allow quick exploration of trends.
